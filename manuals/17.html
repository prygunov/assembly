Команды CLC, CMC, STC

Команды CLC, CMC и STC - это команды для операций над флагом переноса CF. С помощью этих команд можно установить флаг в 1, сбросить его в 0, либо изменить его значение на противоположное.

 

Команда CLC
Синтаксис: CLC
Операнды: Нет
Назначение: Сброс флага переноса
Процессор: 8086+
Флаги: Сбрасывается флаг CF
Комментарий: Команда CLC очищает флаг CF. На другие флаги или регистры она не влияет.
Ограничения: Нет
Примеры: 
clc
Команда CMC
Синтаксис: CMC
Операнды: Нет
Назначение: Инвертирование флага переноса
Процессор: 8086+
Флаги: Значение флага CF изменяется на противоположное
Комментарий: Команда CMC изменяет на противоположное значение флага CF. На другие флаги или регистры влияния не оказывает.
Ограничения: Нет
Примеры: 
cmc
Команда STC
Синтаксис: STC
Операнды: Нет
Назначение: Установка флага переноса
Процессор: 8086+
Флаги: Устанавливается флаг CF
Комментарий: Команда STC устанавливает флаг CF.
Ограничения: Нет
Примеры: 
stc
