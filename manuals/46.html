Команда NOP

Команда NOP - команда "пустышка". Ничего не делает, служит для заполнения места в коде, например для выравнивания его по границам параграфов.

 

Команда NOP
Синтаксис: NOP
Операнды: Нет
Назначение: Нет операции
Процессор: 8086+
Флаги: Не изменяются
Комментарий: Команда NOP не выполняет никаких действий. Это однобайтовая команда, которая занимает место в памяти, но не влияет на состояние процессора, за исключением указателя команд (IP).
Команда NOP представляет собой мнемонический псевдоним команды XCHG AX,AX
Ограничения: Нет
Примеры: 
nop
